# JavaScript Offline Chatbot
It's a client side chatbot created using elasticlunar plugin.

## Features Included :

- JS light weight chatbot
- Offline chatbot
- JSON Knowledgebase 
- Interactive chat UI
- Usage of (elastilunar) powerful text search library
